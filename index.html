Navegador

Un navegador web es un software que permite el acceso a la web, interpretando la informacion de distintos tipos de archivos, y mostrandola al internauta de forma adecuada.

Frontend

Es la parte del desarrollo web que se dedica a la parte frontal de un sitio web.

HTML

Lenguaje de Marcado de Hipertexto: define una estructura basica y un codigo para la definicion de un contenido de un pagina web, como texto, imagenes, videos, juegos, entre otros.
